# C++ Learning MIPT

This repository contains C++ homework solutions and learning materials from MIPT (Moscow Institute of Physics and Technology).

## üìÅ Structure

The project is organized into chapters, each containing specific topics and homework solutions.

### [Chapter 1](./chapter1)
- **Topics**: `nullptr_t`, volatile, pointer safety.
- **Files**:
    - `HW1_1_solution.cpp`: nullptr_t type conversions and volatile qualifiers.
    - `HW1_2_solution.cpp`: Valid null pointer dereferencing scenarios.
    - `HW1_3_solution.cpp`: Pointer safety and protection mechanisms.

### [Chapter 2](./chapter2)
- **Topics**: C++23 features, standard modules.
- **Files**:
    - `CPP23_Explanation.md`: Explanation of C++23 features.
    - `std_module_test`: Tests for standard modules.

## üöÄ Getting Started

### Prerequisites
- C++ compiler compatible with C++11 or later (C++23 for Chapter 2 features).
- CMake (optional).

### Compilation
Navigate to the respective chapter or file to compile.

Example for Chapter 1:
```bash
cd chapter1
clang++ HW1_1_solution.cpp -o HW1_1_solution
./HW1_1_solution
```
